## Services

- **Gas Appliance Installation** — gas stoves, burners (2, 3 & 4-burner) and water heaters, installed by certified technicians. Every job is completed with a Certificate of Compliance (CoC).
- **Gas Cylinder Refills** — full range from 3kg to 48kg, for domestic, hospitality and industrial use.
- **Gas Tank Transportation** — road tanker delivery of any volume of LPG, to any location, anywhere in Africa.

## About this site

This is a single-page static website (`index.html`) — no build tools or dependencies required. It's hosted for free using GitHub Pages.

### Hosting / updating the site

1. Edit `index.html` directly in this repository (or upload a new version via "Add file" → "Upload files").
2. Go to **Settings → Pages** and confirm the source is set to the `main` branch, `/root` folder.
3. Changes go live automatically within a minute or two of being pushed.
